## Backend Contact Server
To enable contact form email delivery:

1. Copy `.env.example` to `.env`.
2. Fill in your SMTP provider details and the recipient email.
3. Run `npm install`.
4. Start the server with `npm start`.

The portfolio will be served at `http://localhost:3000`, and the contact form will send messages to your configured email address.
